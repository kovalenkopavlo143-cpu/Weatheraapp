# WeatherToday 
Wetter-App in Echtzeit.

### Funktionell:
* Wettersuche nach Stadtnamen.
* Dynamisch wechselnde Symbole je nach Wetterlage (Sonne, Regen, Wolken).
* Anzeige von Windgeschwindigkeit und Luftfeuchtigkeit.

### Tools:
Sprachen: HTML5, CSS3, JavaScript (ES6)

Tools & APIs: OpenWeather API, Git, VS Code, Google Fonts
